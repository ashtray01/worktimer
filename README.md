# worktimer
routine manager in terminal
<br />простой таймер задач в терминале
<br />
<br />Вводи задачи (для завершённых задач начни с '!')
<br />Команды:
<br />:home :h — завершить день
<br />:s :save — сохранить в Excel
<br />:q :quit :exit — выйти
